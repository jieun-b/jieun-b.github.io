## 문제

[14503**번: 로봇 청소기**](https://www.acmicpc.net/problem/14503)

### 문제 요약

로봇이 다음 규칙에 따라 청소를 수행함:

1. 현재 칸이 **아직 청소되지 않았다면 청소**
2. **현재 방향 기준으로 왼쪽부터 차례로 탐색**하며,
    - 아직 청소되지 않았고 벽이 아닌 곳이 있다면 그 방향으로 회전하고 한 칸 전진
    - 네 방향 모두 청소되어 있거나 벽이면 **한 칸 후진**
3. 후진도 불가능(벽이면) → 작동 종료

## 풀이 아이디어

### 1. 상태 표현

- `(r, c, d)` : 로봇의 현재 위치와 방향
- `visited[r][c] = 1` → 청소 완료
- `info[r][c] = 1` → 벽

`dir = [(-1, 0), (0, 1), (1, 0), (0, -1)]`

→ 북, 동, 남, 서 순서로 이동 방향을 정의

### 2. 왼쪽 방향으로 회전하며 탐색 (`search` 함수)

```python
next_d = (4 + d - 1) % 4
```

- 왼쪽으로 회전한 방향(`next_d`)을 구함
- 해당 방향으로 이동 가능한지 확인:
    - 지도 범위 안이고(`0 <= r < n, 0 <= c < m`)
    - 아직 청소 안 됐고(`visited == 0`)
    - 벽이 아닌 곳(`info == 0`)

### 3. 후진 처리

```python
prev_r, prev_c = cur_r - dir[cur_d][0], cur_c - dir[cur_d][1]
```

현재 바라보는 방향의 **반대쪽으로 한 칸 후진**

- 후진 자리가 벽이라면 종료 (`info[prev_r][prev_c] == 1`)
- 아니면 위치만 이동 (방향 그대로 유지)

## 코드

```python
import sys
input = sys.stdin.readline

n, m = map(int, input().split())
r, c, d = map(int, input().split())
info = [list(map(int, input().split())) for _ in range(n)]
visited = [[0]*m for _ in range(n)]

dir = [(-1, 0), (0, 1), (1, 0), (0, -1)]

def search(r, c, d): 
    for _ in range(4):
        next_d = (4+d-1)%4 
        next_dir = dir[next_d] # (1, 0)
        next_r, next_c = r+next_dir[0], c+next_dir[1]
        if 0<=next_r<n and 0<=next_c<m and visited[next_r][next_c] == 0 and info[next_r][next_c] == 0:
            return [next_r, next_c, next_d]
        else:
            d = next_d
    return []

cur_r, cur_c, cur_d = r, c, d
count = 0
while(True):
    if visited[cur_r][cur_c] == 0:
        visited[cur_r][cur_c] = 1
        count += 1
    res = search(cur_r, cur_c, cur_d)
    if res:
        cur_r, cur_c, cur_d = res[0], res[1], res[2]
        continue
    else:
        prev_r, prev_c = cur_r-dir[cur_d][0], cur_c-dir[cur_d][1]
        if info[prev_r][prev_c] == 1:
            break
        else:
            cur_r, cur_c = prev_r, prev_c
print(count)
```
