## 문제

[Level2: 모음사전](https://school.programmers.co.kr/learn/courses/30/lessons/84512)

### 문제 요약

- 사용 가능한 문자는 **모음 5개(A, E, I, O, U)**
- 만들 수 있는 모든 **길이 1~5 단어**가 사전에 등재
- 사전은 `"A" → "AA" → ... → "UUUUU"` 순으로 정렬
- 입력 단어 `word`가 **사전에서 몇 번째인지** 찾아서 반환

## 풀이 아이디어

- DFS로 **길이 1~5까지 모든 단어 생성**해 `dictionary`에 저장
- 생성된 단어 목록은 사전 순서 그대로 쌓임
- `dictionary.index(word) + 1` → 해당 단어의 사전 내 순서 반환

## 코드

```python
def solution(word):
    answer = 0
    
    dictionary = []
    alphabets = ['A', 'E', 'I', 'O', 'U']
    
    def dfs(cur):
        if len(cur) == 5:
            return
        for alphabet in alphabets:
            dictionary.append(cur+alphabet)
            dfs(cur+alphabet)
            
    dfs('')
    
    answer = dictionary.index(word) + 1
    return answer
```
