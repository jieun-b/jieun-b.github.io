## 문제

[Level2: 의상](https://school.programmers.co.kr/learn/courses/30/lessons/42578)

### 문제 요약

- 주어진 의상들로 만들 수 있는 **서로 다른 착용 조합의 수**를 구하는 문제
- 같은 종류의 의상은 **1개만 착용 가능**
- 다른 종류의 의상은 자유롭게 조합 가능
- **최소 1개는 반드시 착용**

## 풀이 아이디어

- 종류별로 의상 개수 세기
- 각 종류는 “입기/안 입기” 선택이 있으므로 `(개수 + 1)`을 모두 곱하기
- 마지막에 **아무것도 안 입는 경우** 1가지 제외하기

## 코드

```python
from collections import defaultdict
 
def solution(clothes):
    answer = 1
    info = defaultdict(int)
    for cloth in clothes:
        info[cloth[1]]+=1
    for cnt in info.values():
        answer *= (cnt+1)
    return answer - 1
```
