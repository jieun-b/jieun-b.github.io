## 문제

[Level3: 여행경로](https://school.programmers.co.kr/learn/courses/30/lessons/43164#)

### 문제 요약

- **ICN에서 출발**하여 주어진 **모든 항공권을 정확히 한 번씩** 사용해 경로를 만드는 문제
- 가능한 경로가 여러 개면 **사전순으로 가장 빠른 경로**를 선택

## 풀이 아이디어

- 항공권을 간선으로 보고 DFS로 **모든 티켓을 사용한 경로** 탐색
- 경로를 만들 때 **티켓별 visited 체크**로 중복 사용 방지
- 티켓들을 **사전순으로 정렬해 둔 뒤**, 사용하지 않은 티켓만 선택해 다음 공항으로 이동
- 티켓 모두 사용하면 경로를 저장 → **사전순 첫 번째 경로 반환**

## 코드

```python
def solution(tickets):
    answer = []
    
    tickets.sort()
    visited = [False]*len(tickets)
    
    def dfs(start, path):
        if len(path)-1 == len(tickets):
            answer.append(path[:])
            return
        for i in range(len(tickets)):
            if not visited[i] and tickets[i][0] == start:
                visited[i] = True
                path.append(tickets[i][1])
                dfs(tickets[i][1], path)
                path.pop()
                visited[i] = False
        
    dfs('ICN', ['ICN'])
    return answer[0]
```
