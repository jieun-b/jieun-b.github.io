## 문제

[13335**번: 트럭**](https://www.acmicpc.net/problem/13335)

### 문제 요약

- 모든 트럭이 다리를 건너는 데 걸리는 시간 구하기
- 트럭들이 순서대로 다리를 건너야 하며, 트럭의 무게는 서로 같지 않을 수 있음
- 다리의 길이는 `w`, 견딜 수 있는 최대 하중은 `L`
- 다리 위에는 최대 `w`대의 트럭이 동시에 올라갈 수 있지만, 그 트럭들의 무게 합은 `L`을 넘을 수 없음
- 한 번에 1초마다 트럭이 한 칸씩 이동

## 풀이 아이디어

### 1. 큐(`deque`)를 이용한 시뮬레이션 구조

- **`queue`** → 아직 다리에 오르지 않은 트럭들의 무게를 저장
- **`bridge`** → 현재 다리 위 상태를 저장
    - 길이는 항상 `w` (다리 길이)
    - 다리 위를 “트럭의 위치”로 표현 (`0`은 빈칸)

```python
queue = deque(a)          # 대기 중인 트럭들
bridge = deque([0] * w)   # 다리 위 (길이 w)
```

### 2. 매초 반복

- 다리 위 이동 (맨 앞 트럭 내리기)
    - 한 칸 앞으로 이동하는 효과 = `popleft()`
    - 다리의 맨 앞 트럭이 빠짐 → 다리 끝을 벗어난 것
- 새로운 트럭이 올라갈 수 있는지 확인
    - 남은 트럭이 있고(`if queue:`), 현재 다리 위 트럭 무게 + 새 트럭 무게 ≤ 하중(`L`)이면 트럭을 올림
    - 그렇지 않으면 빈칸(`0`) 추가
- 트럭이 한 칸씩 이동할 때마다 1초 증가

## 코드

```python
import sys
from collections import deque
input = sys.stdin.readline

n, w, l = map(int, input().split())
a = list(map(int, input().split()))

queue = deque(a)
bridge = deque([0]*w)
count = 0
while(bridge):
    bridge.popleft()
    if queue:
        if sum(bridge)+queue[0] <= l:
            bridge.append(queue.popleft())
        else:
            bridge.append(0)
    count += 1
print(count)
```
